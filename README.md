# MacQish

### Author  : @LuciMadMax

##### MacQish is Linux Based MacSpoofing Tools

##### Setup

###### git clone https://github.com/DedSec-OS/MacQish

###### cd MacQish

###### sudo python3 setup.py


##### Use

###### macqish  -i  interface  -m  new mac  or  auto

##### example:

###### macqish -i wlan0 -m 0c:1d:22:3d:12:0b

###### macqish -i wlan0 -m auto

###### macqish -a auto
